# 新路由 3（Newifi-D2）自动编译最新版 OpenWrt（Lede）

_每 12 小时自动编译一次，在 Action 中下载_

## 特性

- 主题: argon
- 开源 WIFI 驱动/闭源驱动同时编译一次
- 外挂存储:exfat, ntfs, ext4

## 插件

- aliddns
- aria2
- easymesh(开源驱动版有)
- eqos // IP 网速控制
- frpc // 内网穿透客户端
- helloworld
- hd-idle // 硬盘休眠
- mwan3/mwan3helper/syncdial // 均衡负载及多拨
- openvpn/openvpn-server
- qos
- samba // Samba 文件共享
- ttyd //网页终端
- turboacc // Turbo ACC 网络加速
- vlmcsd // KMS 服务器
- wolplus
- wrtbwmon
- xlnetacc
- zerotier

## 网关地址

- IP 192.168.1.1
- Username root
- Password password
